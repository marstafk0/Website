My Projects

Vending Machine 

The Vending Machine Web Application uses REST, JQuery, Ajax, HTML, CSS, and JavaScript. The website performs transactions with the user, from adding money to the virtual wallet, to seeing consistently updated stock, price, and items from a server application pulled from a database. Other functions include perfect error handling and dispensing JSON change data back to the user. 

DVD Library Application

Not my most impressive, here I developed a  DVD Library Application to track titles, publishers, directors, notes, dates and ratings, searchable by title or ID. Handles all errors and exceptions succinctly, with a full range of unit tests using JUnit, Java in a MVC design pattern. 

Flooring Ordering App

Here I have a Flooring ordering program that creates, reads, updates and deletes orders respectively. Uses dependency injection to keep all classes working independent of each other, cohesively and efficiently. Not to mention all the tests, exceptions etc. This one I'm more proud of than others, I'll probably end up turning this into a full fledged web app.

ToDo Spring Application

This is a Spring Boot Application called ToDo Application that allows users to use PostMan to create, retrieve all, retrieve by ID, update or delete to do tasks which are pulled from MySQL using JDBC. This one was a fun one to do.

Hotel Reservation Database

This is a Hotel Reservation Database in SQL Server using TSQL to keep track of hotel information including room numbers, prices, reservations with customer info, as well as reservation dates start to end, adults, children, and amenities. All of which easily query-able.
